# 模型属性

## User
    uuid        string  max=64      primary key         # id主键
    nickname    string  max=128                         # 昵称
    avatar      string  max=512                         # 头像url
    address     Address                                 # 地址
    phone       string  max=11      unique              # 手机号
    email       string  max=128     unique              # 邮箱号(辅助找回密码)
    password    string  max=128                         # 密码
    age         int     >0 <120                         # 年龄
    tags        [Tag]                                   # 标签(口味偏好)
